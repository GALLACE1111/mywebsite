如何用 SQL 確認資料是否正確寫入

開啟 MySQL 命令列

mysql -u root leaderboard_db

常用查詢指令

查看排行榜

SELECT * FROM user_total_loves ORDER BY total_loves DESC;

查看所有用戶

SELECT * FROM users;

查看分數記錄

SELECT * FROM love_scores ORDER BY created_at DESC LIMIT 10;

查看特定用戶的總分

SELECT u.username, t.total_loves FROM user_total_loves t JOIN users u ON t.user_id = u.id WHERE t.user_id = 1;

提示：提交分數後，可以用 SQL 查詢確認資料是否正確寫入資料庫

建議流程：提交分數 → 用 SQL 查詢 → 再次獲取排行榜確認
