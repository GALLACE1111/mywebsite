# 背景音樂資料夾

請將您的音樂檔案放在這個資料夾中。

## 使用方式

1. 將音樂檔案（mp3格式）放入此資料夾
2. 將檔案命名為 `bgm.mp3`
3. 或者修改 `script.js` 中的音樂路徑

## 建議的音樂類型

- **戰鬥BGM**: 節奏快、充滿動感的音樂
- **休閒BGM**: 輕鬆、舒適的背景音樂

## 音樂設定

預設音量：30%
您可以在 `script.js` 中修改 `musicVolume` 變數來調整音量（0.0 - 1.0）

## 範例檔名

- `bgm.mp3` - 預設音樂檔案
- `battle.mp3` - 戰鬥音樂（備選）
- `chill.mp3` - 休閒音樂（備選）
